ProofLink — Payroll ↔ Recordkeeper Reconciliation Reference Implementation

Reference implementation / IP asset for sale or license. ProofLink is a pre-revenue, IP-only reference implementation for automated reconciliation and compliance verification in 401(k) retirement plans. The codebase compares payroll contribution data against recordkeeper transaction data to detect discrepancies, timing violations, and regulatory compliance issues. This is a complete, functional codebase ready for integration into a buyer's platform.

What It Includes

  • Preflight validation gate: Enforces explicit mapping contract and required field validation before reconciliation runs
  • Canonical mapping system: Vendor-agnostic column mapping with exact-match normalization (no fuzzy logic)
  • Core reconciliation engine: Automated comparison of payroll vs recordkeeper contributions
  • Compliance analyzers: Secure 2.0 catch-up rules, IRS 402(g) limits, eligibility drift detection, timing violation checks
  • Evidence generation: Audit-ready evidence packs with detailed exception reports
  • Vendor detection: Automatic identification and normalization for multiple payroll systems (ADP, Paychex, Paylocity, etc.)
  • Test suite: Smoke tests with sample data demonstrating preflight gate functionality
  • Documentation: Complete schema documentation, mapping examples, and buyer handoff guides

What It Does Not Include

  • Hosted service or SaaS platform
  • Data adapters for specific payroll or recordkeeper systems (buyer must build these)
  • Fuzzy ingestion or probabilistic matching (strict exact-match + normalization only)
  • Production deployment infrastructure
  • Ongoing support or maintenance (asset sale only)
  • Commercial usage history (pre-revenue prototype)
